# Campus-Connect-Directory

**Campus Connect Directory**  is  a client-side web application designed to help students browse and search university clubs and events in an interactive, user-friendly way. It provides a dynamic single-page experience using only HTML, CSS, and JavaScript.  

## Project Overview

This project demonstrates proficiency in core web development fundamentals while delivering an interactive experience. Students can view, filter, and search through club and event information without page reloads.  

Key focus areas include:  
- **HTML & CSS**:  Structuring and styling a clean, responsive interface using Flexbox/Grid  
- **JavaScript**:  Handling dynamic data rendering, DOM manipulation, and interactive filtering  
- **UI/UX**:  Ensuring a professional and responsive design for desktop and mobile users  

## Planned Features

1. **Data Structure**:  Store club and event details as a JavaScript array of objects  
2. **Dynamic Rendering**:  Generate club and event cards dynamically on page load  
3. **Search & Filtering**:  Real-time search bar and filter options for easy navigation  
4. **Detailed View**:  Display detailed information for selected clubs/events without reloading the page  
5. **Responsive Design**:  Ensure optimal display on all devices using modern HTML/CSS techniques
